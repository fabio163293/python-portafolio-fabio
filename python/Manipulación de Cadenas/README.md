Día 18 - Manipulación de Cadenas

Fecha: 09 de mayo de 2025

Descripción (ES/EN)

Español:Este día se enfoca en aprender a manipular cadenas (strings) en Python, una habilidad clave para procesar texto. Se incluyen ejercicios prácticos para formatear texto, buscar palabras y reemplazar términos, aplicando manejo de excepciones para garantizar robustez.  

English:This day focuses on learning to manipulate strings in Python, a key skill for text processing. It includes practical exercises for formatting text, searching for words, and replacing terms, applying exception handling for robustness.
Estructura

ejercicio_1_formateador.py: Formatea un texto ingresado por el usuario, capitalizando cada palabra y eliminando espacios extras.  

ejercicio_2_buscador.py: Busca una palabra en una frase, mostrando su primera posición y cantidad de apariciones.  

ejercicio_3_reemplazador.py: Reemplaza una palabra en un texto y guarda el resultado en resultado.txt.
