# Día 6 - Estructuras de Datos y Entrada/Salida de Archivos

## Descripción
Este día se centró en el manejo de estructuras de datos como listas y diccionarios, junto con la lectura y escritura de archivos de texto en Python. Se trabajó de manera práctica desarrollando scripts que permiten almacenar información, procesarla y persistirla en archivos externos.

## Ejercicios realizados

### 1. **Registro de Calificaciones de Estudiantes**
- Se solicita la cantidad de estudiantes.
- Se almacena el nombre y la calificación de cada uno.
- Se valida que la calificación esté entre 0 y 100.
- Los datos se guardan en el archivo `calificaciones.txt`.

### 2. **Lectura de Archivo de Productos**
- Se crea un archivo `productos.txt`.
- Se desarrolla un lector de archivos que imprime cada línea sin caracteres de salto de línea.

### 3. **Creador de Diccionario desde Archivo**
- Se lee `calificaciones.txt`.
- Cada línea (clave: valor) se convierte en un par clave-valor de un diccionario en Python.
- Se usó `ast.literal_eval` para convertir strings de listas a listas reales.

### 4. **Creación de Tabla Separada por Comas**
- Se crea un archivo `crear_tablas.txt` usando `csv.writer`.
- Se escribieron filas simulando un pequeño dataset.
